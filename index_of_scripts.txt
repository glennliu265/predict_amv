A List of Scripts for Predict AMV

An attempt to document the scripts for this project. Started 2023.04.04, adding scripts as I write them.


Symbology

(x) --> consider moving to scrap
(!) --> major work needed/unfinished



# -----------
## Parameters
# -----------
    train_cesm_params       :: Parameter dictionary for each training run, accessed by experiment name
    predict_amv_params      :: General parameters for the project (plotting, names, paths, etc)

# ----------------
## Preprocessing
# ----------------

    regrid_reanalysis_cesm1 :: (!)      Regrid reanalysis datasets (HadISST) to CESM1 resolution using xesmf_env 
    prep_data_reanalysis    :: (x)(!)   Preprocess reanalysis datasets regridded by CDO (wip)
    regrid_ocean_variables  :: Regrid ocean variables in CESM1
    make_landice_mask       :: Make land-ice (and pacific) mask, cropped to region

# ----------------
## Models
# ----------------
    test_reanalysis         :: Test NN trained on CESM1-LE on HadISST
    train_NN_cesm1          :: Main NN Training Script for CESM1-LE data

# ----------------
## Analysis
# ----------------

    calc_amoc_max                 :: Compute Index of AMOC strength and examine the autocorrelation/predictability timescale of individual members
    test_predictor_uncertainty    :: Test a model trained on one predictor using another, different predictor.
    test_cesm_witheld             :: Visualize frequency count by year for members withheld from CESM1 training (based on test_reanalysis_data.py)
    viz_acc_by_predictor 	  ::  Visualize accuracy by leadtime and predictor


# ----------------
## Modules
# ----------------
    amvmod                        :: All functions for the project
    amv_dataloader                :: Functions for loading processed data. Includes path names, etc
    pamv_visualizer               :: Convenience functions for visualization

# ------------------
## Current Workflow (from raw data)
# ------------------


(1) Preprocessing
    >> Regrid Ocean Variables    :: regrid_ocean_variables
    >> Make Land/Ice Mask        :: make_landice_mask
    >> Compute QNet              :: calc_nhflx_predictor
    >> Preprocessing             :: prep_data_byvariable

   	>> Combine variables and crop to time/region
    	>> Compute Annual mean
    	>> Standardize
    >> Compute Index             :: prepare_regional_targets




(2) Training

    - train_NN_cesm1 		:: Main NN Training Script for CESM1-LE data

(3) Analysis

    - viz_acc_by_predictor 	:: Visualize accuracy by leadtime and predictor
    - test_reanalysis 		:: Test NN trained on CESM1-LE on HadISST
